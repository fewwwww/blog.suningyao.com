жҗӯе»әFisco BsoCеҢәеқ—й“ҫзҪ‘з»ң(ARMжһ¶жһ„macOSе№іеҸ°)

рҹҳ… TLDR: Fisco BsoCиІҢдјјжҡӮдёҚж”ҜжҢҒArmжһ¶жһ„зҡ„macOS (Fri Jun 25 2021 13:56:05 GMT+0800 (CST))

1. жҗӯе»әеҚ•зҫӨз»„FISCO BCOSиҒ”зӣҹй“ҫ#

жң¬иҠӮдҪҝз”ЁејҖеҸ‘йғЁзҪІе·Ҙе…·build_chain.shи„ҡжң¬еңЁжң¬ең°жҗӯе»әдёҖжқЎ4иҠӮзӮ№зҡ„FISCO BCOSй“ҫ, д»ҘmacOS arch64 11.4зі»з»ҹдёәдҫӢж“ҚдҪң.

е®ҳж–№жҺЁиҚҗдёӢиҪҪйў„зј–иҜ‘зЁӢеәҸ. з”ұдәҺжҳҜARMжһ¶жһ„,еӣ жӯӨйңҖиҰҒзј–иҜ‘жәҗз ҒиҺ·еҸ–fisco-bcosеҸҜжү§иЎҢзЁӢеәҸ

еҰӮжһңйҖҡиҝҮйў„зј–иҜ‘зЁӢеәҸиҝӣиЎҢжҗӯе»ә, з”ҹжҲҗзҡ„FISCOй“ҫж—¶дјҡжҳҫзӨә: [ERROR] We only offer x86_64 precompiled fisco-bcos binary, your OS architecture is not x86_64. Please compile from source.

з¬¬дёҖжӯҘ: е®үиЈ…дҫқиө–#

йҖҡиҝҮHomebrewиҝӣиЎҢдҫқиө–зҡ„е®үиЈ….

brew install openssl git flex bison gmp cmake

з”ұдәҺжңүдәӣдҫқиө–еңЁжӣҙж–°еүҚдёҚж”ҜжҢҒARMзҡ„macOS,дјҡжҳҫзӨә: Error: git: no bottle available!

жүҖд»ҘйңҖиҰҒе…ҲиҝҗиЎҢbrew updateжқҘиҺ·еҸ–жңҖж–°еҸ‘еёғзҡ„ARMзүҲжң¬дҫқиө–, еҶҚиҝӣиЎҢе®үиЈ…

з¬¬дәҢжӯҘ: е…ӢйҡҶжәҗз Ғ#

# иӢҘзҪ‘з»ңиүҜеҘҪgit clone https://github.com/FISCO-BCOS/FISCO-BCOS.git
# иӢҘеӣ дёәзҪ‘з»ңй—®йўҳеҜјиҮҙй•ҝж—¶й—ҙж— жі•жү§иЎҢдёҠйқўзҡ„е‘Ҫд»Ө, иҜ·е°қиҜ•дҪҝз”Ёgiteeжәҗ:git clone https://gitee.com/FISCO-BCOS/FISCO-BCOS.git

з”ұдәҺж–Үд»¶иҫғеӨ§, дёӢиҪҪиҖ—ж—¶еҸҜиғҪжңүдәӣй•ҝ.

з¬¬дёүжӯҘ: иҝӣиЎҢзј–иҜ‘#

вҡ пёҸ: д»ҺиҝҷдёҖжӯҘејҖе§Ӣ, з”ұдәҺcmakeи„ҡжң¬жңӘйҖӮй…ҚARMжһ¶жһ„Macbook, еӣ жӯӨж— жі•жӯЈеёёиҝҗиЎҢ.

cd FISCO-BCOSmkdir -p build && cd buildcmake -DARCH_NATIVE=on ..make -j4# еҰӮжһңзј–иҜ‘иҝҮж…ў, еҸҜд»ҘжӣҝжҚўдёәmake -j10еҗҢж—¶ејҖеҗҜеҚҒдёӘд»»еҠЎеҠ йҖҹжһ„е»ә

зј–иҜ‘е®ҢжҲҗеҗҺдәҢиҝӣеҲ¶ж–Үд»¶дҪҚдәҺFISCO-BCOS/build/bin/fisco-bcos.

дҫқиө–еә“иҫғеӨ§, жңү400Mе·ҰеҸі, еҰӮжһңеӣ дёәзҪ‘з»ңй—®йўҳеҜјиҮҙй•ҝж—¶й—ҙж— жі•дёӢиҪҪдҫқиө–еә“

иҜ·е°қиҜ•йҖҡиҝҮ git clone https://gitee.com/FISCO-BCOS/LargeFiles.git жүӢеҠЁдёӢиҪҪдҫқиө–еә“пјҢе°Ҷ/libsеҶ…ж–Үд»¶ж”ҫеңЁFISCO-BCOS/deps/src/
