ERCж ҮеҮҶд»ҘеҸҠNFTеә”з”ЁејҖеҸ‘ рҹҺЁ

дёҖ. ERCж ҮеҮҶ#

1. д»Җд№ҲжҳҜERC?#

ERCжҳҜжҢҮд»ҘеӨӘеқҠе·ІжӯЈејҸеҢ–зҡ„жҸҗжЎҲ, ERC-721е…¶е®һе°ұд»ЈиЎЁ721еҸ·жҸҗжЎҲ. йҖҡиҝҮеҗ„з§ҚжөҒзЁӢзҡ„е®Ўи®®дёҺжөӢиҜ•еҗҺ, дёҖдёӘйқһжӯЈејҸзҡ„EIPе°ұеҸҜд»ҘиҪ¬еҸҳжҲҗERC, жҲҗдёәж ҮеҮҶеҢ–зҡ„еҚҸи®®.

д»ҺејҖеҸ‘иҖ…зҡ„и§’еәҰжқҘиҜҙ, ERCе°ұжҳҜйңҖиҰҒе®һзҺ°зҡ„дёҖдәӣжҺҘеҸЈ. иҝҷдәӣжҺҘеҸЈзҡ„еә•еұӮе®һзҺ°еҸҜд»ҘеҸӮиҖғOpenZeppelin

еңЁ EIP-1 дёӯе…·дҪ“йҳҗжҳҺдәҶEIPзҡ„дҪңз”ЁдёҺз”ҹе‘Ҫе‘Ёжңҹ. е…·дҪ“е®Ўи®®жөҒзЁӢеҰӮдёӢ:

иҝ‘жңҹжӯЈеңЁжөӢиҜ•зҪ‘з»ңдёӯйғЁзҪІзҡ„ EIP-3074 (ж–°еўһдёӨдёӘж–°зҡ„жҢҮд»Ө)дёҺ EIP-1559 (и°ғж•ҙдәӨжҳ“иҙ№з”Ёжһ„жҲҗ), еҰӮжһңйҖҡиҝҮдәҶжөӢиҜ•е’Ңе®Ўж ёе°ұдјҡжӯЈејҸжҲҗдёәERC.

ж ҮеҮҶеҢ–жҸҗжЎҲзҡ„дҪңз”Ёе°ұжҳҜз»ҷејҖеҸ‘иҖ…дёҖдәӣжЁЎзүҲеё®еҠ©ејҖеҸ‘, еҗҢж—¶д№ҹз»ҷејҖеҸ‘иҖ…дёҖдәӣж ҮеҮҶзҡ„йҷҗеҲ¶.

  • ејҖеҸ‘иҖ…дёҚз”Ёд»Һеә•еұӮејҖе§Ӣе®Ңж•ҙиҝӣиЎҢи®ҫи®ЎдёҺе®һзҺ°, иҖҢеҸӘйңҖж №жҚ®дёҖдәӣж ҮеҮҶеҝ«йҖҹејҖеҸ‘DApp. еҰӮд»ҠеҖҹеҠ©иҝҷдәӣж ҮеҮҶ, з”ҡиҮіжңүдәӣзҪ‘з«ҷеҒҡеҲ°еҸӘйңҖиҰҒеЎ«4дёӘж•°еҖје°ұеҸҜд»Ҙе®һзҺ°еҸ‘еёҒзҡ„еҠҹиғҪ.

  • ејҖеҸ‘иҖ…еҸҜд»ҘйҒөеҫӘиҝҷдәӣж ҮеҮҶ, иҝҷж ·иғҪеҒҡеҲ°ж•ҙдёӘз”ҹжҖҒзҡ„дә’йҖҡд»ҘеҸҠе…¶д»–иҪҜд»¶йҖӮй…ҚдёҠзҡ„дҫҝеҲ©. жҜ”еҰӮ: MyEtherWalletдёҚз”Ёйў‘з№Ғжӣҙж–°е°ұеҸҜд»Ҙж”ҜжҢҒеӨҡдёӘеёҒз§Қ.

2. жңүе“ӘдәӣERC?#

д»ҘдёӢжҳҜйғЁеҲҶдјҡиў«йў‘з№ҒдҪҝз”ЁеҲ°зҡ„ERCж ҮеҮҶ.

еҗҚз§°еҶ…е®№
ERC-20еҸҜжӣҝжҚўиө„дә§зҡ„еҺҹе§Ӣд»ЈеёҒеҗҲзәҰ, дҫӢеҰӮ: EOS, OMG.
ERC-223е…је®№ERC-20,дҝқжҠӨжҠ•иө„иҖ…д»ҘйҳІж„ҸеӨ–зҡ„еҗҲзәҰиҪ¬иҙҰ
ERC-721*йқһеҗҢиҙЁд»ЈеёҒпјҲNFTsпјүж ҮеҮҶпјҢеҸҜдҪңдёәдә§жқғиҝӣиЎҢдәӨжҳ“

ERC-918еҸҜејҖйҮҮжҖ§д»ЈеёҒпјҢе…Ғи®ёеҠ е…ҘжҢ–зҹҝз®—жі•

ERC-1400иҜҒеҲёйҖҡиҜҒж ҮеҮҶпјҢйғЁеҲҶеҸҜдә’жҚўд»ЈеёҒпјҢиҜҘEIPж ҮеҮҶе…·жңүиғҪеҠӣиҝӣиЎҢејәеҲ¶иҪ¬з§»

Fun fact: еңЁжҹҘиҜў д»ҘеӨӘеқҠEIPе®ҳж–№ж–ҮжЎЈ ж—¶, еҸҜд»ҘеҸ‘зҺ°GitHubд»“еә“еҶ…зҡ„ж–Үд»¶жҳҜжҢүз…§ еӯ—з¬ҰдёІйЎәеәҸжҺ’еәҸ зҡ„, еӣ жӯӨEIP-2жҺ’еңЁEIP-1996д№ӢеҗҺ, жЈҖзҙўж—¶йңҖиҰҒжіЁж„Ҹ.

ERC-20:#

еҗҢиҙЁеҢ–д»ЈеёҒзҡ„еҹәзЎҖж ҮеҮҶ, еңЁ2015е№ҙиў«VitalikжҸҗеҮә, ж—ЁеңЁж ҮеҮҶеҢ–дёҖдәӣеёёи§ҒеҺ»дёӯеҝғеҢ–еә”з”Ё. е…¶дёӯд»–жҺўи®ЁдәҶеҮҪж•°зҡ„еҸҳйҮҸе‘ҪеҗҚи§„еҲҷ, д»ҘеҸҠеҗҢиҙЁеҢ–д»ЈеёҒзҡ„дёҖдәӣеҝ…иҰҒж–№жі•е’ҢдәӢд»¶.

д»ҘдёӢжҳҜERC-20зҡ„е…·дҪ“еҶ…е®№:

  • name, symbol, decimals иҝҷдёүдёӘж–№жі•йқһеҝ…йңҖ, дҪңз”ЁжҳҜжҸҗй«ҳеҸҜз”ЁжҖ§, дёҺе®һйҷ…еҠҹиғҪж— е…і.

    1. name(): иҝ”еӣһд»ЈеёҒзҡ„еҗҚз§°, жҜ”еҰӮ: Ethereum.
    2. symbol(): иҝ”еӣһд»ЈеёҒзҡ„д»ЈеҸ·, жҜ”еҰӮ: BTC.
    3. decimals(): иҝ”еӣһж•ҙж•°, иЎЁзӨә1дёӘд»ЈеёҒиғҪиў«еҮ дҪҚе°Ҹж•°иЎЁзӨә.

  • ERC-20дёӯзҡ„еҝ…иҰҒж–№жі•.

    1. totalSupply() : иҝ”еӣһд»ЈеёҒзҡ„жҖ»йҮҸ.
    2. balanceOf(): иҫ“е…ҘиҙҰжҲ·ең°еқҖ, иҝ”еӣһең°еқҖеҶ…зҡ„д»ЈеёҒдҪҷйўқ.
    3. transfer(), transferFrom(): еҝ…йЎ»и§ҰеҸ‘TransferдәӢд»¶(еҢ…жӢ¬ж•°йўқдёә0). д»ҺжҖ»йҮҸдёӯиҪ¬иҙҰз»ҷиҙҰжҲ·ең°еқҖ/д»ҺдёҖдёӘиҙҰжҲ·иҪ¬иҙҰз»ҷеҸҰдёҖдёӘиҙҰжҲ·.
    4. allowance(): еңЁиҝӣиЎҢиҪ¬иҙҰж—¶, жҹҘиҜўеү©дҪҷеҸҜиҪ¬иҙҰйўқеәҰ.
    5. approve(): и®ҫзҪ®ж¶Ҳиҙ№иҖ…д»ЈеёҒиҪ¬з§»йўқеәҰйҷҗеҲ¶, жҲҗеҠҹеҗҺи§ҰеҸ‘ApproveдәӢд»¶. еҪ“йҮҚеҶҷйҷҗеҲ¶ж—¶, еҝ…йЎ»е…ҲйҮҚеҶҷдёә0, еҶҚи®ҫзҪ®зңҹе®һеҸӮж•°, д»ҘйҒҝе…Қзұ»дјјж”»еҮ».

    4. еңЁд»ҳж¬ҫж–№ең°еқҖдёӯеҮҸеҺ»иҪ¬иҙҰйҮ‘йўқ
    5. еңЁж”¶ж¬ҫж–№ең°еқҖдёӯеҠ дёҠиҪ¬иҙҰйҮ‘йўқ

    3. йӘҢиҜҒеү©дҪҷйўқеәҰжҳҜеҗҰи¶іеӨҹжң¬ж¬ЎдәӨжҳ“, дёҚи¶іеҲҷеӣһж»ҡдәӨжҳ“
    4. жӯЈејҸе®ҢжҲҗдәӨжҳ“

  • mint(), burn() д»ЈеёҒеўһеҸ‘й”ҖжҜҒйҖ»иҫ‘:

3. ERC-721 жҰӮиҝ°#

йқһеҗҢиҙЁд»ЈиЎЁзӢ¬дёҖж— дәҢпјҢCryptoKittiesдёәдҫӢ, жҜҸеҸӘзҢ«йғҪжӢҘжңүзӢ¬дёҖж— дәҢзҡ„еҹәеӣ . дёҖеҸӘзҢ«е°ұжҳҜдёҖдёӘNFT. зҢ«е’ҢзҢ«д№Ӣй—ҙжҳҜдёҚиғҪзҪ®жҚўзҡ„. иҝҷз§ҚзӢ¬зү№жҖ§дҪҝеҫ—жҹҗдәӣзЁҖжңүзҢ«е…·жңүж”¶и—Ҹд»·еҖј, д№ҹеӣ жӯӨеҸ—еҲ°иҝҪжҚ§.

  еҫ—еҲ°ж•°жҚ®зҡ„жөҒзЁӢеҰӮдёӢ: йҖҡиҝҮ`Web3`жӢҝеҲ°зҪ‘з»ңID => еҰӮжһң`Color.json`дёӯеӯҳеңЁжӯӨзҪ‘з»ң => жӢҝеҲ°`Color.json`дёӯзҡ„`abi`ж•°жҚ®, жӢҝеҲ°еҗҲзәҰең°еқҖ => жӢҝеҲ°е…·дҪ“еҗҲзәҰеҶ…е®№дёҺж•°жҚ® => жӢҝеҲ°NFTжҖ»йҮҸдёҺж•°жҚ®
- еңЁйЎөйқўдёӯжёІжҹ“жҲ‘д»¬жӢҝеҲ°зҡ„еҗ„дёӘNFT
    ```html    ...           </div>        <hr/>        {/* йҖҡиҝҮеҲ—иЎЁеҪўејҸжёІжҹ“жүҖжңүзҡ„color, жҜҸдёӘйўңиүІжҳҜcolorдёӯеЈ°жҳҺзҡ„йўңиүІ */}        <div className="row text-center">          { colors.map((color, key) => {            return(                <div key={key} className="col-md-3 mb-3">                  <div className="token" style={{ backgroundColor: color }}></div>                  <div>{color}</div>                </div>            )          })}        </div>    ...    ```  еңЁ`App.css`дёӯж·»еҠ ж ·ејҸ, и®©еҗ„дёӘNFTжӣҙеҠ зҫҺи§Ӯ    ```css    .token {        height: 150px;        width: 150px;        border-radius: 50%;        display: inline-block;    }    ```

    - е®ҢжҲҗжҲ‘д»¬зҡ„Webз«Ҝиҫ“е…Ҙе№¶жҸҗдәӨзҡ„еҠҹиғҪ
