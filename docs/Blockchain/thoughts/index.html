💡 关于区块链的一些个人思考

搞区块链就是为了来赚钱/洗钱的. 不忘初心, 牢记使命.

投研方面一些小小的工作

2022/10/27: 以下内容其实不是最重要的.

You see, cryptography is the only technology available to any individual today that enables defence against even the most powerful adversaries (like a mighty totalitarian regime).

Privacy is a tautological requirement in the sovereign individual thesis.

  1. 投资加密货币确实是投机, 但投资也是投机.

  2. 比特币不值钱, 美元也不值钱.

  3. DeFi目前还是是新壶装旧酒的庞氏.

  4. 做支付/稳定币的区块链项目都对外宣称自己是为了经济自由度.

  5. 对比特币价格的共识 !== 比特币内的共识机制, 数字人民币 !== 支付宝/微信支付, 数字人民币App用户使用上约等于支付宝/微信支付.

  6. 中国的场景(高并发, 移动支付, 央行)是最适合研究数字货币的. 央行把活儿分散给到了商业银行, 方便央行的制度管理和政策调整, 商业银行的活动开展和运营管理.

巧合的是, 我在大一全年都是线上授课, 因此从未当面见到老师和同学, 恰巧写了关于人的精神自我和物理自我的关系的文章. 这和元宇宙也有所关系, 感兴趣的话可以阅读一下. 大二也水了一篇关于去中心化的, 这篇有点瞎写.

  1. 加密游戏的重点是游戏, 而不是加密, 加密的部分大多只是激励机制. (211104: 这条不对, 重点就是币. 211225: 游戏要么就PPT, 专注于搞钱; 要么就搞得真的很好玩, 不然不上不下很没意思.)

  2. 如果一定要投资加密货币, 那么只需要买入比特币, 其他的诸如以太坊或者其他垃圾币都是中心化的💩.

  3. 只有比特币和以太坊的设计 值得研究, 其他的都是拙略的模仿者.

  4. 都怪预言机.

  5. MetaVerse就像Teenage Sex, 每个人都在讨论它, 却没人知道怎么做, 每个人都觉得别人已经做过了, 所以每个人都声称自己也做过.

  6. 人与其他生物的区别就是思考与创造.

  7. 我们从罗马、希腊到互联网的创造过程中就一直在尝试去中心化, 虽然一直在失败.

  8. 社会问题, 信任问题, 权力分配问题, 人文思想, 人类文明, 数字艺术, 去中心化, 创作的欲望, 思考的欲望.

  9. 传统游戏: 游戏 -> 价值; NFT: 价值 -> 游戏.

  10. 经济系统是区块链项目吸引人的本源, 没有币就没有人会去免费宣传项目和布道.

  11. 国内的区块链项目不能沾币了, 就相当于打UFC却只能用拳头.

  12. 一味追求去中心化是没用的, 为了弥补去中心化性能的不足, 要么公司出钱, 要么用户出钱, 最后都是用户出钱, 用户不会高兴.

  13. 用户就是上帝, 甭管项目多牛逼, 用户不认, 项目就是💩.

  14. 不要试图说服傻逼, 不要试图让傻逼搞懂区块链是什么, 就像你推荐给小学生世界名著, 在小学生没有人生阅历和知识储备的情况下是不可能读懂的.

  15. 大家知道这是庞氏, 知道这是骗局.

  16. 反复翻阅这篇文章, 心怀感恩, 让金钱随着时间推移.

  17. 生于忧患, 死于安乐.

  18. GameFi比DeFi更容易让人接受, 大家知道打游戏得到好装备能卖钱. GameFi更好的是, 传统游戏氪金进去就是黑洞, GameFi还能给你返收益.

  19. 止损不止盈, 不然永远不可能暴富.

  20. 其实上涨靠的基本上都是大趋势carry. 天时地利, 再加上自己踩对时机.

  21. 学而不思则罔, 思而不学则殆.

  22. 保持学习, 保持思考, 保持谦虚.

  23. 可组合性很重要, 就像复利一样可怕. Idea Legos.

  24. DAO 就是白嫖之王. 项目无, 代码无, 白皮书无, 做个官网喊个口号就可以开干.

  25. Web2: 你就是产品; Web3: 产品是你的.

  26. Web1: get data; Web2: get/post data; Web3: get/post/own data.

  27. Pied Piper 牛逼.

  28. 一切为了赚钱. 股权不投, 没有币圈基因. 要花时间精力孵化的不投.

  29. DYOR. 保持学习, 保持好奇心, 保持敬畏心.

  30. 找到属于自己的圈子, 并且要敢于做那个质疑的人.

  31. ARWEAVE YYDS!

  32. 重要的书都应该连着读两遍, 因为第二遍读的时候, 你已经知道结局了, 这样才能真正理解开头. 另一个原因是第二遍阅读时, 你有不一样的心情, 可能会从另一种角度看待问题.

  33. NFT 需要被重新定义, 远比想象得牛逼.

  34. 创作者和开发者是造物主.

  35. 基础设施或许已经过多了.

  36. 创作者和粉丝和中介的关系会被 NFT 重新定义.

  37. 互联网传递信息, 区块链传递价值, 信息与价值是现代战争最重要的无形战场, 甚至可能比物理战争更加重要.

  38. 看这篇我的文章

  39. Arweave: 历史不再由胜利者书写, 而是 Arweave 书写. 同时 Arweave 上的 SCP 代表了 Web3 无限的可能性.

  40. Social FUCKING Alpha!!!

  41. Innovation is innovation. 把所有所七八糟的扯的东西上链是很无聊的, 做一个Web3版本的某某东西是很无聊的. Superfluid, AsyncArt, RSS3, Arweave, Uniswap, Ethereum, BTC (货币不由胜利者发行) 这些才算是真正的 Innovation 和 Reinvention. Dfinity, Filecoin 这种真的比较没意思. 无论如何, 拉的和辣的 Innovation 现在都发生在 Crypto 领域, 这很好.

  42. Innovation === Innovation

  43. 要看一个生态是否真的有人玩, 就看是否有真正有社群的 NFT 项目, 目前来看就只有以太坊和 Solana 满足.

  44. 现在的 zk 都是 validity proof, 假 zk, 没隐私. 很多术语都需要更新.

  45. 看 1000 个, 研究 100 个, 聊 10 个, 投 1 个.

  46. 设计上已经过度安全, 实践中却无法匹配设计.

  47. NFT 跨链桥的缺失主要是因为没有良好的 NFT 定价系统, 预言机喂不过来.

  48. 跨链桥不敢无脑用预言机, 因为很多时候它们自己的体量比 Chainlink 大.

  49. 社交 NFT = 奢侈品, 收藏品 NFT = 艺术品.

  50. 一句话说不完的项目不是好项目. 简化, 讲清楚到底项目优化了哪个变量.

  51. L1 和 L1 跨不了主要是因为各自有不同的共识.

  52. Cosmos 的卖点其实就是 App as Chain.

  53. Rollup 类的 Layer3 appchain 部署麻烦, subnet 部署方便, 共享安全性, 但是不 scale.

  54. NFT 对艺术品的投资会越来越大, 因为暴富之后还是想混 Old Money 圈子.

  55. 用户是爹, 从用户角度考虑.

  56. 对于 MEV 来说, 如何捕捉 MEV 比抗 MEV 更有吸引力.

  57. 去中心化的存在是否需要, 很重要.

  58. Rollup 的信任模型就是一个智能合约, 这很牛逼.

  59. 更好的 Web > 更好的区块链.

  60. zk Rollup 中心化其实够了, 理由参见 zkvm 那一篇末尾的补充.

  61. wen web3 accessibility?

  62. 对 web3 产品来说, 我从没看过收入什么 PX 比什么的, 就是卖币, 没了.

  63. Arweave vs Filecoin = Optimistic Rollup vs zk Rollup = 奖励好学生 vs 惩罚坏学生 = cryptoeconomic vs cryptography

  64. Professor making lots of great points on rollups and this

  65. 走到后面才意识到原来当初为啥那么设计.

  66. 不看且放弃的几个领域: 隐私、音乐 NFT、AI、比特币相关、小游戏、硬件.

  67. 三权分立, 开发立法, DAO裁决, 网络行政. 但是团队通过代币分配和主动权绑架了权力, 上演多数人暴政. 只有创始人跑路了的比特币杜绝了这个问题, 削弱了开发团队的能力.

  68. 比特币永远滴神.

  69. 从社会的发展上来看, MEV(neo-liberalism)和DAO(美国社会)的治理问题无法解决.

  70. NFT 的唯一性不存在于元数据和智能合约, 而是存在于智能合约的 ID 上或者说独特的区块空间.

  71. 目前 Web3 只有基础设施, 没有任何应用.

  72. Crypto 该做的是 target 更多技术宅, 目前别奢望 mass adoption. 毕竟是从黑客开始的.

  73. 基础设施 => 应用? => 基础设施? => 应用?

  74. zk 无法完全解决问题, 一旦问题不被完全解决, 那么就等于没有被解决.

  75. 技术方案和架构和代码是个屁, 解决问题和创造价值最重要.

  76. 共享安全性没用, 共了半天 trust model 还没 rollup 稳.

  77. 波卡和 Cosmos 一直都不是跨链, 而是多链.

  78. zk 的核心价值在于证明和验证的解耦, 这很模块化.

  79. zk 原理 (本来想写下的, 但是几句话说不清楚, 核心图).

  80. Still early. Few.

  81. 看山是山 (这个东西真好玩, 好多知识, 颠覆金融), 看山不是山 (体量太小了, 总归要走向中心化, 性能垃圾, 骗局, 迟早失败), 看山是山 (求索的过程才重要, 拥抱泡沫, 寻找价值). 没懂知道自己不懂, 没懂感觉懂了, 真的懂了.

    1. 知道方向, 2. 知道信息在哪里, 3. 知道信息的知识.

  83. 任何事情都讲究新手福利期 (最有想象力和最能成长和摸索清楚事情怎么做的阶段), 要习惯从零到一.

  84. 我的 bookmark 里有 1000 篇文章, 每个研究员都需要有一个图书馆.

  85. TWAP 完全展现了区块链是一个时钟.

  86. 反人性的活动 (跑步), 可以通过代币来激励.

  87. PoS 让随机性下降了, 需要防范一些更多层面的攻击.

  88. 链上验证 === on-chain 哈哈...

  89. sovereign rollup 不如 pos 和 secured rollup, 无法 onchain 验证某些事情, 性能瓶颈可能在 settlemenmt layer.

  90. sovereign rollup 比 pos 好的就是不用 validator set, 可以直接 composable, 比 secured rollup 更好就是可以随意选择合约的环境. “可组合性”

  91. 比特币的脚本没有 loop, 因为都只是针对 utxo 解锁这个功能的.

  92. 比特币的智能合约是扶不起的阿斗.

  93. 稳定币本质上是指数基金.

  94. vc 不 care 去中心化.

  95. 不要轻易相信任何人, 他们可能有很好的理由, 结果是为了割你...

  96. 项目就是 move faster, or die slowly.

  97. 做局做局!

  98. 当心抄袭狗.

  99. 养社区最重要.

  100. 多去侧面打听, 越多的信息代表着越多越多的信息. 正好 100, 懒得继续整理了, 你还是直接看我文章吧...
