一天内组建一个研究 DAO 🤝

受南京大学的教授之托, 帮忙组建了一个用 GitHub 和 Notion 管理的学术论文写作 DAO. 大概花了两天时间就建起来了. 以下是步骤:

  1. 理清楚要干的事情, 交流需求.

  2. 写宣传页面, 弄个 Markdown, 介绍介绍, 加个联系方式就行. 最好中英文.

  3. 准备单独的研究项目的介绍页面.

  4. 发布悬赏, 中英文.

  5. 建一个 Notion 协作区以供写作.

  6. 来人了就先单独聊聊, 然后分配悬赏任务, 拉进 Notion.

OK, 这样就在至多两天内把教授想弄的研究 DAO 弄完了.
