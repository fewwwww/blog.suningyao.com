💗 我喜欢的我自己的歌

以时间顺序, 从晚到早, 阐述和排列.

感兴趣的话可以直接到我的网易云听: msfew.

msfew - rnd.​

全名叫 random 嘛, 一首凌晨做的 demo, 尝试一下很难的唱功和 rnb.

msfew - radio.​

很慵懒的哈.

msfew - 420.​

这个突然发现歌词很好...

msfew - airdrop.​

这个也很好听, 但是需要一个甜美的女声来翻唱一下, 重制一下.

msfew - zara.​

这个大家特别喜欢.

msfew - 0616.​

这个我感觉调音调的最正常的一首.

msfew - ok.​

这首比较搞怪逗趣的风格.

msfew - kkp.​

z? kkp. 这首被网易云官方下架了, 歌词太那个...

msfew - princess.​

这个电的比较舒服的, rnb.

msfew - gonsa.​

这个非常浑然天成, 音乐天赋, 华语情歌王子.

msfew/Albedok - glow.​

verse1 是我.

msfew/Albedok - right.​

verse1 是我. 这个是去录音棚的地铁上写的曲和词.
