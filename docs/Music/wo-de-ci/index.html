🐲 解读龙胆紫的《我的瓷》

最近几天一直在听龙胆紫的我的瓷. 作为一个说唱爱好者及蹩脚歌手, 我来斗胆过度分析一下我的瓷的词.

// 我的瓷, 或者我的词. 谐音梗.
《我的瓷》

// 约等于说话不经过大脑, 虎得很.
我的瓷 什么都不想 没有不敢说的
// 没太理解... 阳光宅男是吧.
我的瓷 宅在家 能把全世界都乐了
// 走到哪里算哪里, 还是虎.
我的瓷 出道 从来 不问地儿
// 出手干脆.
我的瓷 出手 从来 不问事儿
// 从小就牛逼.
我的瓷 不会变 从小 就内样儿
// 我的瓷们.
穷的穷 横的横 称的称 怂的怂
// 搏击俱乐部的第一条规矩.
我的瓷 有规矩 规矩都不用说
// 一眼万年.
我的瓷 看眼神就能知道谁是自己人
// 还是下手利落.
我的瓷 脑子快 下手 不落空
// 人狠话不多.
胆儿肥 心细 办正经事儿不淡逼
// 太有钱了, 但是还是睡沙发, 对比.
我的瓷 睡沙发 零花钱能买你家
// 太穷了, 数学题有点喜剧.
我的瓷 穷疯了 过日子像数学题
// 哥们一直底层.
我的瓷想好 我的瓷可好不了
// 时间顺序...
戴手套 干一票 戴头套 戴手铐
// 我的词, 太燥.
我的瓷 太燥 阴三儿 龙胆紫
// 哥们真上法制进行时了.
我的瓷这不是综艺这是法制进行时

// 凶猛.
我的瓷 从小为了兄弟就打架不要命
// 威猛, 强硬.
个个比刘华强还猛 比向华强还硬
// 穷, 但正直.
扎的浑身都花了照样心比钱还干净
// 不怕死.
不怕死不怕折 就为了信的直站的正
// 还是人狠话不多.
不说话 上去就抡
// 张嘴就是大事.
一张嘴就让你丫闷
// 没懂这句.
我的瓷 天天束着
// 天天玩非法物品是吧.
我的瓷 天天晕
// 有福同享, 有难同当.
在一块喝自来水 到现在 喝香槟
// 哥们玩得都太危险了.
我的瓷 死的死折的折活着就不省心
// 同年同日死.
我的瓷 要走一块走 有钱一块花
// 赌性很大.
就像赢把大的子弹一大把 好坏一把抓
// We are family 是吧.
我的瓷 身后一大家子流着一样的血
// 胡同儿文化.
我们欻这一片儿甩你们丫这条街
// 这句开玩笑的.
我的瓷 开奔驰 开玩笑 开公司
// 包小姐姐.
包房 包场 包你们想都不敢想
// 就是直.
我的瓷 看不惯的人和事儿可真不惯着
// 架着.
把我干歪了我的兄弟照样能让我站着

// 都看鸡巴了, 可想而知多么瓷.
我的瓷 鸡巴红
// 这要死.
狂犬病 都不稳定
// 跳楼是吧.
运动员 都往下蹦
// 飞行员是吧.
艺术家 都往上送
// 确实野.
路子野 闯大祸
// 这个没特别懂.
不弄别的圈踢带着解说操你们大爷
// 这个不懂.
职业码逼码
// 收保护费呢.
拽一电话敲家门 今儿上房顶给你丫折腾塌了聊你妈呢
// 上操无视纪律, 下操接受管教.
我的瓷 上操在主席台下了课在办公室
// 马牛逼敢吃屎. 胆子大.
打个石膏比划中指 我的瓷 敢吃屎
// 地为枕, 天为被.
躺担架 躺公园 躺沙发 躺地板
// 瘾太大.
看一电影半盒烟冲这事儿他应该躺宫殿
// 您猜怎么着, 死了.
我的瓷 躺在棺材里
// 死了.
时间紧没赶上新出的游戏机
// 死了, 年岁没长.
给哥哥们躺在前面现在成了我弟
// 艺术家 27 死的定律.
我的瓷我穿着你的盔甲活了一二十七
